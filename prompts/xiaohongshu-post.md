## Prompt 1：小红书爆款标题生成器 | Viral Title Generator

**难度 Difficulty:** ⭐ 简单 Easy  
**适用 AI:** Claude / ChatGPT / Gemini  
**用途 Use Case:** 生成吸引眼球的小红书标题 | Generate eye-catching Xiaohongshu titles

### 🇨🇳 中文 Prompt

```
我需要为小红书帖子写[数字]个吸引眼球的标题。

主题：[填入主题，例如：如何用AI每天多赚500元]
目标受众：[填入受众，例如：25-35岁想尝试副业的上班族]

标题要求：
1. 包含数字或具体结果
2. 引发好奇心或情绪
3. 15字以内
4. 符合小红书风格（真实、分享、有用）
5. 带上相关emoji

请列出[数字]个标题选项。
```

### 🇬🇧 English Prompt

```
I need [number] eye-catching titles for a Xiaohongshu post.

Topic: [Fill in topic, e.g., How to earn extra $500/month with AI]
Target audience: [Fill in audience, e.g., office workers aged 25-35 looking for side income]

Title requirements:
1. Include specific numbers or results
2. Trigger curiosity or emotion
3. Under 15 characters (Chinese) / under 60 characters (English)
4. Match Xiaohongshu style (authentic, sharing, helpful)
5. Include relevant emojis

Please list [number] title options.
```

### ✅ 使用示例 | Example Output

| 输入 Input | 输出 Output |

| 主题：AI副业 | ✨用AI副丛，我小月多草2000元 |

## Prompt 2：小红书完整图文文案 | Full Post Copywriter

**难度 Difficulty:** ⭐⭐ 中级 Intermediate  
**适用 AI:** Claude (推荐) / ChatGPT  
**用途 Use Case:** 生成完整的小红书图文帖子 | Generate complete Xiaohongshu posts

### 🇨🇳 中文 Prompt

```
你是一个小红书爆款内容创作专家。请帮我写一篇小红书帖子。

主题：[填入主题]
我的身份：[填入身份，例如：AI工具教学博主]
目标受众：[填入受众]

请按照以下结构写作：

🔖 标题：（吸引眼球，带emoji）

📝 正文：（200-300字）
- 开头：用真实场景引共鸣
- 中间：提供3个实用建议，每个加一个emoji
- 结尾：召唤评论或协动

🏷️ 标签：（20-30个相关标签）
```

### 🇬🇧 English Prompt

```
You are a Xiaohongshu viral content expert. Write a complete post for me.

Topic: [Fill in topic]
My identity: [Fill in, e.g., AI tools educator]
Target audience: [Fill in audience]

Please write using this structure:

🔖 Title: (eye-catching, with emoji)

📝 Body: (200-300 Chinese characters)
- Opening: Hook with a relatable real scenario
- Middle: 3 practical tips, each with an emoji
- Ending: Call to action for comments or interaction

🏷️ Hashtags: (20-30 relevant tags)
```

### ✅ 小红书平台特点 | Platform Tips

- 观众属性：25-40岁女性为主，关注财富管理、AI工具、提升自己
- 内容风格：第一人称分享，真实经历，有螎入感
- 关键元素：封面图吸睛 + 标题吸眼 + 数据/故事开头
- Audience: Women 25-40, interested in wealth, AI tools, self-improvement
- Style: First-person sharing, authentic experience, relatable
- Key elements: Eye-catching cover + hook title + data/story opening
