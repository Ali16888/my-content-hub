# YouTube Script Prompt Template

## 我的频道定位
- 目标受众：华人中产阶级企业主、想用AI提升效率的人群
- 内容方向：AI工具教学 + 财务自由理念
- 语言：普通话为主（70%），英语（30%）
- 风格：专业但亲切，实用性强，有行动步骤

## YouTube脚本生成提示词

请用以下格式帮我写一篇YouTube视频脚本：

**主题：** [在这里填入主题]

### 脚本结构：
1. **开场钩子（0-30秒）**：用一个痛点问题或震惊数据吸引观众
2. **自我介绍（30秒-1分钟）**：简短介绍我是谁，这个视频能给观众什么价值
3. **内容主体（1-8分钟）**：分3-5个核心要点，每个要点配实际例子
4. **行动呼吁（最后1分钟）**：告诉观众下一步该做什么
5. **结尾（最后30秒）**：订阅提醒 + 下期预告

### 要求：
- 用口语化中文，避免书面语
- 每个要点控制在2分钟内
- 加入具体数字和案例
- 结尾要有订阅/点赞引导
- 总时长目标：8-12分钟
