## Prompt 1：YouTube 完整视频脚本生成器 | Full Video Script Generator

**难度 Difficulty:** ⭐⭐ 中级 Intermediate  
**适用 AI:** Claude (强烈推荐) | Claude (Highly Recommended)  
**用途 Use Case:** 生成完整YouTube视频脚本 | Generate complete YouTube video scripts

### 🇨🇳 中文 Prompt

```
你是一个专业YouTube视频脚本写作专家。请帮我写一个完整的视频脚本。

视频主题：[填入主题]
目标观众：[填入观众，例如：想学AI工具的中国创业者]
视频时长：[填入，例如：8-10分钟]
我的风格：[填入，例如：亲切实用，教学风格]

请按照以下结构写作：

　90秒开头》（立刻抓住观众）
- 第1句：震撼开场或引发深思的问题
- 主要承诺：这个视频将让你学会...
- 订阅引导

　90秒第一部分标题》
- 主题介绍与背景
- 为什么重要

　90秒第二部分标题》
- 核心内容第一部分

　90秒第三部分标题》
- 核心内容第二部分

　90秒结尾》
- 总结要点
- 行动号召（订阅+评论）
- 预告下期内容

每个部分包含：具体台词、过渡语、配合B-Roll的提示。
```

### 🇬🇧 English Prompt

```
You are a professional YouTube scriptwriter. Please write a complete video script for me.

Video topic: [Fill in topic]
Target audience: [Fill in, e.g., Chinese entrepreneurs who want to learn AI tools]
Video length: [Fill in, e.g., 8-10 minutes]
My style: [Fill in, e.g., friendly and practical, educational]

Please write using this structure:

[90-Second Hook]
- First sentence: powerful opener or thought-provoking question
- Main promise: what viewers will learn from this video
- Subscribe CTA

[Section 1 - 90 seconds]
- Topic intro and background
- Why it matters

[Section 2 - 90 seconds]
- Core content part 1

[Section 3 - 90 seconds]
- Core content part 2

[90-Second Outro]
- Summary of key points
- Call to action (subscribe + comment)
- Preview of next video

Each section should include: specific dialogue, transitions, B-Roll suggestions.
```

## Prompt 2：YouTube 标题 + 封面文字 | Title & Thumbnail Copy

**难度 Difficulty:** ⭐ 简单 Easy  
**适用 AI:** Claude / ChatGPT  
**用途 Use Case:** 优化视频标题和封面文字，提升点击率

### 🇨🇳 中文 Prompt

```
我正在制作一个关于[主题]的YouTube视频。

请帮我生成：

1. 　10个YouTube标题》
   - 包含关键词，适合SEO
   - 引发点击欲望
   - 60字符以内

2. 　5个封面文字方案》
   - 简短有力，5字以内
   - 适合封面图展示
   - 能和图片相道而驰

视频主题：[填入主题]
目标观众：[填入观众]
```

### 🇬🇧 English Prompt

```
I'm creating a YouTube video about [topic].

Please generate:

1. [10 YouTube Titles]
   - Include keywords for SEO
   - Trigger click-through desire
   - Under 60 characters

2. [5 Thumbnail Text Options]
   - Short and powerful, under 5 words
   - Suitable for thumbnail display
   - Complements the visual

Video topic: [Fill in topic]
Target audience: [Fill in audience]
```
